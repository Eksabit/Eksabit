Переменные в Python

Переменные — это именованные области памяти, которые используются для хранения данных. В Python переменные могут хранить различные типы данных, и их можно использовать для выполнения операций и манипуляций с этими данными.

Объявление переменных

В Python переменные объявляются просто путем присвоения значения. Например:

name = "Alice"
age = 30
is_student = True

В этом примере мы объявили три переменные: name, age и is_student.

Типы переменных

Python поддерживает несколько основных типов данных, которые можно использовать в переменных:

  • Целые числа (int): Целые числа, например, 10, -5.
  • Числа с плавающей запятой (float): Числа с десятичной точкой, например, 3.14, -0.001.
  • Строки (str): Последовательности символов, заключенные в одинарные или двойные кавычки, например, "Hello", 'World'.
  • Логические значения (bool): Значения True или False.
Примеры использования переменных

Давайте рассмотрим несколько примеров использования переменных в коде:

1. Целые числа
num1 = 10
num2 = 5
sum_result = num1 + num2
print(f"Сумма: {sum_result}")

В этом примере мы объявили две переменные num1 и num2, сложили их и вывели результат.

2. Строки
first_name = "Alice"
last_name = "Smith"
full_name = first_name + " " + last_name
print(f"Полное имя: {full_name}")

Здесь мы объединили две строки, чтобы получить полное имя.

3. Логические значения
is_raining = False
if is_raining:
    print("Не забудьте взять зонт!")
else:
    print("Погода хорошая!")

В этом примере мы используем логическую переменную is_raining для проверки, идет ли дождь.

Изменение значений переменных

Переменные в Python могут изменять свои значения в процессе выполнения программы. Например:

count = 0
count += 1
print(f"Текущее значение: {count}")

Здесь мы увеличиваем значение переменной count на 1 и выводим его.

Заключение

Переменные являются основным строительным блоком программирования в Python. Они позволяют хранить и манипулировать данными, что делает код более гибким и мощным. Понимание работы с переменными и их типами — это важный шаг на пути к освоению Python.
